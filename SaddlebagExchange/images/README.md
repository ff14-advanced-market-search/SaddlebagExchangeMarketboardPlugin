# Plugin icon (D17 and /xlplugins)

**`icon.png`** in this folder is the plugin icon (chocobo with saddlebag) used in `/xlplugins` and for D17 submission. It must be square, 64×64–512×512 px (512×512 recommended).

When you open a PR to DalamudPluginsD17, the folder you add must include `manifest.toml` and `images/icon.png`.
