## AI Use Disclosure

**Level: Co-Pilot** — AI tools were used on demand code generation and bug finding.

### Code Generation
- **Cursor** and **GitHub Copilot** were used for some code generation. All AI-generated code was reviewed, tested, and validated by the human developer.

### Testing & Quality Analysis
- **Codacy**, **SonarCloud**, and **CodeRabbit** were used for static analysis, bug detection, and code quality review. These tools assisted in identifying issues; a human developer reviewed and resolved findings.

### Human Involvement
- Plugin was **human-designed** and **human-reviewed**
- Portions of the code were **written by hand**
- Personally **tested** prior to submission
